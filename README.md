# API Subservice - TTS Conversion

## Introduction

It's a sub API for SkillSharp, which use in Scenario and Interview features. It used: 
- **Edge TTS** to convert text to sound.
- **Rhubarb Lip Sync** to generate lip-sync JSON data.
